# HTML File Index: Web-2-Notion

This document provides an overview of all HTML (.html) files in the Web-2-Notion extension, with a brief explanation of each file's purpose.

---

## Root Directory

- **icon_generator.html**: Tool for generating extension icon images in various sizes and formats.
- **site-selectors.html**: UI for managing custom site selectors, allowing users to define rules for content extraction.
- **test-page.html**: Test page for development and debugging extension features.

---

## Subfolders

### highlightTooltip/

- **highlightTooltip.html**: The HTML structure for the highlight tooltip UI, used for annotating and highlighting selections.

### modal/

- **modal.html**: The modal dialog's HTML structure, used for settings, selectors, and user prompts.

### onboarding_guide_popup/

- **index.html**: The onboarding guide popup's HTML, shown to new users for setup and guidance.

### popup/

- **index.html**: The main popup window for the extension, providing quick access to core features.

### restricted_popup/

- **index.html**: The restricted popup UI, shown in limited or special contexts (e.g., when permissions are restricted).

### toast/

- **toast.html**: The HTML structure for toast notifications, used to display brief messages to the user.

### welcome/

- **welcome.html**: The welcome page shown to new users or after installation.

---

## Notes

- Some additional HTML files may exist in other subfolders or as part of build output, but the above list covers all main UI and source files.
- For more details on custom selectors, see `.guides/Custom Site Selectors Guide.md`.
